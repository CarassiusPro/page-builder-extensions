Manage Beaver Builder extensions with this plugin

== Description ==

This plugin features a simple interface to manage extensions for Beaver Builder

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. View the sample_plugin.php file for the code to use in your own plugin


== Frequently Asked Questions ==

= Does this plugin require Beaver Builder to work? =

No, but it is built for plugins andd modules that extend the power of Beaver Builder.

= Can I use this for my own custom modules? =

Absolutely! There are hooks to add new tabs and fields.
